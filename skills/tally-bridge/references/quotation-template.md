## 1. Scope of Work

JICATE Solutions will set up and configure the Tally Bridge service, which includes:

- Installation of the Bridge software on your designated computer
- Provisioning of a secure cloud account and database
- Deployment of a cloud dashboard for real-time voucher data access
- Automated sync configuration (incremental sync every 60 seconds during business hours)
- Initial data migration of all vouchers for the current financial year
- Auto-start setup so the bridge runs on computer boot
- Nightly reconciliation for full data integrity
