# SDG Intelligence

**AI-powered SDG mapping and accreditation report generation for Higher Education Institutions.**

Maps your institution's data (courses, research, outreach, operations) to all 17 UN Sustainable Development Goals and generates ready-to-use reports for NAAC, NIRF, THE Impact Rankings, and QS Sustainability.

## What the Client Gets

| Report Type | What It Contains |
|-------------|-----------------|
| **SDG Scorecard** | All 17 SDGs mapped with strength tiers (Strong/Moderate/Weak/None) |
| **NAAC Attributes 7/9/10** | Ready-to-paste sections for Self-Study Report |
| **NIRF SDG Data** | Formatted data for NIRF submission |
| **THE Impact Rankings** | Mapped data for Times Higher Education |
| **QS Sustainability** | Mapped data for QS rankings |
| **Gap Analysis** | What's missing + specific recommendations to improve |

## How It Works

```
Input:  Institutional data (CSV/Excel — courses, research, outreach, operations)
           ↓
Map:    AI maps each data point to relevant SDGs
           ↓
Score:  Strength tiers — Strong (>60%), Moderate (30-60%), Weak (10-30%), None (<10%)
           ↓
Output: 6 report types as PDF + Markdown, ready for submission
```

## Commands

| Command | What it does |
|---------|-------------|
| `new-client` | Register institution |
| `setup {id}` | Ingest and validate CSV data |
| `audit {id}` | AI maps data to SDGs, generate scorecard |
| `report {id}` | Generate specific report type |
| `verify {id}` | Prove end-to-end delivery |

## Delivery Models

- **One-time (PaP)** — Complete SDG mapping + all reports
- **Managed Operations (MO)** — Annual retainer with quarterly updates and gap evolution tracking
