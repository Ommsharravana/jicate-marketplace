# SEO + GEO + AEO Audit

**Complete website health report with AI visibility analysis, competitor benchmarking, and action plans.**

Not just traditional SEO. This audits how your site performs across three dimensions:
- **SEO** — How Google sees you (PageSpeed, structure, content)
- **GEO** — How AI models (ChatGPT, Gemini, Perplexity) see you (Generative Engine Optimization)
- **AEO** — Whether AI assistants cite you as an answer (Answer Engine Optimization)

## What the Client Gets

A professional PDF report with:
- **Digital Health Score** (0-100) across all three dimensions
- **Per-site scorecards** with specific metrics
- **Competitor comparison matrix** — how you stack up
- **Prioritized action plan** — what to fix first, with specific instructions

## How It Works

```
Input:  Website URL + 2-3 competitor URLs
           ↓
Auto:   Firecrawl (site crawl) + PageSpeed + AI model testing
           ↓
Score:  SEO score + GEO score + AEO score = Digital Health Score
           ↓
Output: PDF report with competitor matrix + prioritized fixes
```

## Commands

| Command | What it does |
|---------|-------------|
| `new-client` | Register client with their URLs |
| `run {id}` | Execute full audit pipeline |
| `deliver {id}` | Generate and deliver PDF report |
| `verify {id}` | Prove end-to-end delivery |
